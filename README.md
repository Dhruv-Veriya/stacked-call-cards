# stacked-call-cards

📇 Call UI Card Manager

A simple JavaScript card management UI that allows users to create and manage contact call notes with a stacked card interface. The project uses HTML, CSS, and Vanilla JavaScript and stores data using LocalStorage so that cards persist even after page reload.

Users can add new call entries through a modal form, and the cards are displayed in a stacked UI layout where they can be navigated using previous and next buttons.

✨ Features

📌 Add new contact cards using a modal form

💾 Data stored in LocalStorage

🃏 Dynamic stacked card UI

⬆️⬇️ Navigate cards using previous and next buttons

🖼 Profile image support via URL

🎨 Clean and minimal interface

🛠 Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

LocalStorage API

📋 Card Information

Each card contains:

Profile Image

Full Name

Home Town

Purpose of Call

Category (Emergency, Important, Urgent, No Rush)

⚙️ How It Works

User clicks the + button to open the form.

User fills in the details and submits the form.

Data is stored in LocalStorage.

Cards are dynamically generated and displayed in a stacked layout.

Users can navigate through cards using ↑ and ↓ buttons.
